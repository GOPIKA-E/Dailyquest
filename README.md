# 🎮 DailyQuest 🚀

Welcome to **DailyQuest** — your daily dose of fun, brainpower, and streak-building addiction! 🔥

Just like your favorite learning apps, DailyQuest keeps you coming back every day with **new challenges**, **game variety**, and a powerful **streak system** that makes skipping a day feel impossible 😈

---

## 🌟 What is DailyQuest?

DailyQuest is a **gamified web app** that gives you a **new interactive challenge every single day**.

🧠 One day you're solving a puzzle  
🔢 Next day you're cracking a math challenge  
🎯 Another day you're testing memory or logic  

All while building your **daily streak** and climbing the **leaderboard** 🏆

---

## 🎮 Features

### 🔥 Daily Challenges
- New challenge every day (auto-rotating)
- Based on real-time date logic
- Never gets boring!

### 🧩 Multiple Game Types
- ✅ Quiz (MCQ)
- 🔤 Word Scramble
- 🔢 Math Challenge
- 🧠 Memory Game
- 🔍 Pattern Recognition
- 🎯 Bonus variations

### 📈 Streak System
- Track your daily consistency 🔥
- Miss a day? Streak resets 😬
- Builds habit & addiction (in a good way!)

### 🏆 Leaderboard
- Save your score
- Compete with others
- See top players

### ⏳ Countdown Timer
- Live timer until next challenge
- Creates urgency & anticipation

### 🎨 Gamified UI
- Neon glowing theme ✨
- Smooth animations
- Hover effects
- Confetti celebration 🎉

### 📱 Mobile Friendly
- Fully responsive design
- Works perfectly on phones

---

## 🛠 Tech Stack

- 💻 HTML
- 🎨 CSS
- ⚙️ JavaScript (Vanilla)
- 🗄 localStorage (for streaks & leaderboard)

No frameworks. No libraries. Just pure frontend magic ✨
